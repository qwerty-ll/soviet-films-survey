# Инструкция по настройке Google Sheets Аналитики

Для того чтобы в вашей таблице появились красивые графики и автоматический анализ ответов, выполните следующие шаги:

### 1. Добавление скриптов в таблицу
1. Откройте вашу Google Таблицу с ответами.
2. Перейдите в меню **Расширения (Extensions)** -> **Apps Script**.
3. В левой панели вы увидите файл `Код.gs` (или `Code.gs`). Убедитесь, что там находится код из файла `google-apps-script/Code.gs`.
4. Нажмите на иконку **"+"** рядом с надписью "Файлы" и выберите **Скрипт (Script)**.
5. Назовите новый файл `Analytics`.
6. Скопируйте содержимое файла `google-apps-script/Analytics.gs` из этого проекта и вставьте его в созданный файл `Analytics.gs` в Google Sheets.
7. Нажмите на иконку дискеты (**Сохранить проект**).

### 2. Запуск аналитики
1. Обновите страницу с Google Таблицей в браузере.
2. В верхнем меню таблицы появится новый пункт: **📊 Аналитика Анкеты**.
3. Нажмите на него и выберите **🚀 Создать полный отчет (Дашборд)**. Это нужно сделать **только один раз**!

### ⚡️ Полная автоматизация:
Я настроил скрипт так, что **вам больше не нужно нажимать кнопку обновления**. 
- Сразу после того, как кто-то заполнит анкету на сайте, скрипт сам пересчитает все данные и обновит графики в фоновом режиме.
- Вы просто открываете лист «Полный Отчет» и видите самые свежие данные в реальном времени.

### Что внутри полного отчета:
- **Все закрытые вопросы:** Скрипт сам определяет тип вопроса (одиночный выбор, множественный или рейтинг) и строит подходящий график.
- **Анализ чекбоксов:** Если в вопросе можно было выбрать несколько вариантов, скрипт разделит их и покажет статистику по каждому пункту отдельно.
- **Группировка:** Все графики выстраиваются в аккуратную сетку, как в профессиональных отчетах.
- **Авто-обновление:** Просто нажмите кнопку снова, если в таблицу упали новые ответы.
